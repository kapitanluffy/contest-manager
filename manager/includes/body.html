Creating Contests

  1. Make sure you have Created Criterias and Added Contestants first
  2. On the Contest Panel, Type an appropriate Contest name (e.g. Beauty Pageant - Elimination)
  3. Select the criterias, percentage must be exactly 100%
  4. Select the contestants
  5. Click 'add'
  6. A message confirming that your contest is added will appear

Connecting Contests

  1. Make sure you have Created the Pre-Contest
  2. On the Contests Panel, Create an appropriate Post-Contest (e.g. Beauty Pageant - Finals) skipping the 4th step
  3. Edit your 'pre-contest' (e.g. Beauty Pageant - Elimination)
  4. On the 'connect to...' drop-box, select your Post-Contest (do not select the same contest)
  5. Enter a number of how many contestants will enter the Post-Contest
  6. Click 'edit'
  7. A message confirming that your contest is updated will appear

Creating Criterias

  1. Type an appropriate Criteria name (e.g. Audience Impact)
  2. Enter a percentage not more than 100%
  3. Click 'add'
  4. A message confirming that your criteria is added will appear

Adding Contestants

  1. Type an appropriate Contestant name (e.g. Shamsey Supsup)
  2. Enter the contestant number
  3. Click 'add'
  4. A message confirming that your contestant is added will appear
